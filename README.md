# Eureka
An idea-generator where people (general public) can post an app idea.

### How to run this locally?
- ``` npm install ```
- ```npm run client-install```
- ```npm run dev```

### keys_dev.js needs to be created in the config folder:
```
module.exports = {
   mongoURI: 'CHANGE_ME',
   secretOrKey: 'secret'
}
```
